
## Run Locally

Clone the project

```bash
  git clone https://github.com/sasananshul/rough-space.git
```

Go inside search-places directory

```bash
  cd search-places
```

Install dependencies

```bash
  npm install
```

Start the dev server

```bash
  npm run dev
```
